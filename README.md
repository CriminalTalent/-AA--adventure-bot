### Monsters 탭

| 컬럼 | 설명 |
|---|---|
| 마물명 | 마물 식별자 ([레이드/마물명] 에서 사용) |
| 장소 | [몬스터목록/장소명] 필터에 사용 |
| 최소나이 | 이 나이 미만 플레이어는 조우 불가 |
| 골드최소/최대 | 성공 시 골드 범위 |
| 대화텍스트 | 조우 연출에 사용되는 마물 대사 |
