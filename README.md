# 무사수행 봇

마스토돈 기반 텍스트 RPG 무사수행 봇입니다.
방향 이동으로 지도를 탐색하며 마물을 조우하고 보물을 발견합니다.
솔로 탐색과 멀티 레이드를 모두 지원합니다.

---

## 시작하기

### 요구 사항

- Node.js 18 이상
- 마스토돈 계정 및 액세스 토큰
- Google Cloud 서비스 계정
- Google Sheets 스프레드시트 (공유 DB)

### 설치
```bash
npm install
```

### 환경 변수
```env
MASTODON_URL=https://your.instance.url
ADVENTURE_TOKEN=your_mastodon_access_token

GOOGLE_SHEET_ID=your_google_sheet_id
GOOGLE_SERVICE_ACCOUNT_JSON={"type":"service_account","project_id":"..."}

MAX_RAID_MEMBERS=4
```

### 실행
```bash
node adventure.js
```

PM2 사용 시:
```bash
pm2 start adventure.js --name adventure-bot
pm2 save
```

## 명령어

모든 명령은 봇 계정을 멘션한 뒤 대괄호 `[ ]` 안에 입력합니다.

@무사수행봇 [명령어]

---

### 솔로 탐색

| 명령 | 설명 |
|---|---|
| `[무사수행]` | 솔로 탐색 시작 (장소 미지정) |
| `[무사수행/장소명]` | 특정 장소로 솔로 탐색 시작 |
| `[북]` `[남]` `[동]` `[서]` | 해당 방향으로 이동 |
| `[귀환]` | 탐색 종료 및 결과 저장 |

솔로 탐색은 unlisted로 게시됩니다 (팔로워 타임라인에만 표시).

---

### 레이드 (멀티)

| 명령 | 권한 | 설명 |
|---|---|---|
| `[레이드]` | 파티장 | 레이드 모집 개설 (마물 랜덤) |
| `[레이드/마물명]` | 파티장 | 특정 마물을 목표로 레이드 개설 |
| `[참가]` | 전체 | 모집 중인 레이드에 참가 |
| `[출발]` | 파티장 | 모집 종료 후 탐색 시작 |
| `[북]` `[남]` `[동]` `[서]` | 파티장 | 해당 방향으로 이동 |
| `[귀환]` | 파티장 | 탐색 종료 및 전원 결과 저장 |

레이드는 퍼블릭으로 게시됩니다 (타임라인 공개).

진행 조건:
- 참가자 전원이 해당 턴 스케줄에 무사수행이 포함되어 있어야 합니다.
- 같은 턴에 중복 진행할 수 없습니다.
- 모집은 개설 후 10분간 유지됩니다.
- 최대 인원은 환경 변수 MAX_RAID_MEMBERS로 설정합니다 (기본값 4명).

---

### 마물 목록 조회

| 명령 | 설명 |
|---|---|
| `[몬스터목록]` | 전체 마물 목록 출력 |
| `[몬스터목록/장소명]` | 특정 장소의 마물만 출력 |

출력 정보: 마물명 / 장소 / HP / 공격력 / 방어력 / 골드 범위 / 설명

## 탐색 시스템

### 이동 및 구역

이동할 때마다 방향별 풍경 묘사가 출력됩니다.
이동 횟수에 따라 구역이 전환되며 마물 강도와 보물 등급이 높아집니다.

| 구역 | 이동 횟수 | 마물 출현율 | 보물 출현율 | 보물 등급 |
|---|---|---|---|---|
| 입구 | 1~3회 | 25% | 20% | 낮음 |
| 외곽 | 4~6회 | 35% | 25% | 중간 |
| 폐허 | 7~9회 | 45% | 28% | 중간 |
| 심부 | 10회+ | 55% | 30% | 높음 |

나머지 확률은 안전 (별다른 이벤트 없음)입니다.

---

### 성공률 계산

솔로: 30 + (체력 x 0.2) + (전투 x 0.5) - (마물 방어력 x 0.3)

레이드: 참가자 전원의 성공률 평균값 적용

하한 10% / 상한 85%

---

### d100 판정

| 구간 | 결과 |
|---|---|
| 1 ~ 성공률 x 0.15 | 대성공 |
| 1 ~ 성공률 | 성공 |
| 성공률+1 ~ 95 | 실패 |
| 96 ~ 100 | 대실패 |

---

### 결과별 효과

| 결과 | 수치 변화 | 골드 |
|---|---|---|
| 대성공 | 전투+4 / 스트레스-3 / 평판+3 / 야망+1 | 마물 평균 골드 x 1.5 (없으면 +250G) |
| 성공 | 전투+2 / 스트레스+2 / 평판+1 | 마물 골드 범위 내 랜덤 (없으면 +100G) |
| 실패 | 체력-4 / 스트레스+4 / 위험도+2 | 없음 |
| 대실패 | 체력-7 / 스트레스+7 / 위험도+4 | -50G |

레이드의 경우 골드는 참가 인원수로 균등 분배됩니다.
수치 변화는 참가자 전원에게 동일하게 적용됩니다.

---

### 보물상자 등급

| 등급 | 상자 종류 | 골드 범위 | 추가 효과 |
|---|---|---|---|
| 낮음 | 낡은 나무 상자 | 30~100G | 없음 |
| 낮음 | 녹슨 철제 상자 | 80~180G | 스트레스-1 |
| 중간 | 잠긴 가죽 상자 | 100~250G | 스트레스-2 |
| 중간 | 문양이 새겨진 상자 | 200~350G | 평판+1 |
| 높음 | 황금 문양 상자 | 250~450G | 평판+1 / 스트레스-2 |
| 높음 | 마력이 깃든 상자 | 350~600G | 야망+1 / 평판+2 |

---

### 강제 귀환

솔로 탐색 중 체력이 5 이하로 떨어지면 자동으로 강제 귀환 처리됩니다.
세션은 마지막 행동 기준 30분간 유지되며, 이후 자동 만료됩니다.
