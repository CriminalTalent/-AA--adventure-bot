# 무사수행 봇

마스토돈 기반 텍스트 RPG 무사수행 봇입니다.
스케줄 봇에서 무사수행을 제출한 플레이어가 이 봇에 멘션하여 전투를 진행합니다.
d100 주사위 판정으로 대성공 / 성공 / 실패 / 대실패 4단계 결과가 출력됩니다.

---

## 시작하기

### 요구 사항

- Node.js 18 이상
- 마스토돈 계정 및 액세스 토큰
- Google Cloud 서비스 계정
- Google Sheets 스프레드시트 (공유 DB)

### 설치
```bash
npm install
```

### 환경 변수
```env
MASTODON_URL=https://your.instance.url
ADVENTURE_TOKEN=your_mastodon_access_token

GOOGLE_SHEET_ID=your_google_sheet_id
GOOGLE_SERVICE_ACCOUNT_JSON={"type":"service_account","project_id":"..."}
```

### 실행
```bash
node adventure.js
```

PM2 사용 시:
```bash
pm2 start adventure.js --name adventure-bot
pm2 save
```

### 무사수행 진행

| 명령 | 설명 |
|---|---|
| `[무사수행]` | 나이에 맞는 마물 중 랜덤 선택하여 진행 |
| `[무사수행/마물명]` | 지정한 마물과 전투 진행 |

진행 조건:
- 해당 턴 스케줄에 `무사수행`이 포함되어 있어야 합니다.
- 같은 턴에 중복 진행할 수 없습니다.
- 지정 마물은 플레이어 나이 이상의 최소나이를 가져야 합니다.

결과는 공개 게시됩니다. DM으로 완료 알림이 발송됩니다.

---

### 마물 목록 조회

| 명령 | 설명 |
|---|---|
| `[몬스터목록]` | 전체 마물 목록 출력 |
| `[몬스터목록/장소명]` | 특정 장소의 마물만 출력 |

출력 정보: 마물명 / 장소 / HP / 공격력 / 방어력 / 골드 범위 / 설명

## 판정 시스템

### 성공률 계산
성공률 = 30 + (체력 × 0.2) + (전투 × 0.5) - (마물 방어력 × 0.3)
- 하한: 10% / 상한: 85%
- 마물을 지정하지 않으면 방어력 페널티 없음

### d100 판정 분기

| 구간 | 결과 |
|---|---|
| 1 ~ 성공률 × 0.15 | 대성공 |
| 1 ~ 성공률 | 성공 |
| 성공률+1 ~ 95 | 실패 |
| 96 ~ 100 | 대실패 |

### 결과별 효과

| 결과 | 수치 변화 | 골드 |
|---|---|---|
| 대성공 | 전투+4 / 스트레스-3 / 평판+3 / 야망+1 | 마물 평균 골드 × 1.5 (없으면 +250G) |
| 성공 | 전투+2 / 스트레스+2 / 평판+1 | 마물 골드 범위 내 랜덤 (없으면 +100G) |
| 실패 | 체력-4 / 스트레스+4 / 위험도+2 | 없음 |
| 대실패 | 체력-7 / 스트레스+7 / 위험도+4 | -50G |

## Google Sheets 연동

### Monsters 탭

마물 데이터를 관리합니다.
마물명 | 장소 | 종류 | 최소나이 | HP | 공격력 | 방어력 | 골드최소 | 골드최대 | 대화텍스트 | 설명

| 컬럼 | 설명 |
|---|---|
| 마물명 | 마물 식별자 (명령어에서 사용) |
| 장소 | `[몬스터목록/장소명]` 필터에 사용 |
| 최소나이 | 이 나이 미만 플레이어는 지정 불가 |
| 골드최소/최대 | 성공 시 골드 범위 |
| 대화텍스트 | 전투 연출에 사용되는 마물 대사 |

### Adventures 탭 (자동 기록)

무사수행 진행 시 자동으로 기록됩니다.
날짜시간 | 플레이어 | 마물 | 결과 | 주사위 | 성공률 | 골드변화 | 잔액

## 플로우
1. 스케줄 봇에 [스케줄/무사수행/...] 포함하여 제출
|
v
2. @무사수행봇 [무사수행] 또는 [무사수행/마물명] 멘션
|
v
3. 성공률 계산 → d100 판정 → 결과 분기
|
v
4. 수치 / 골드 변동 후 Players 탭 업데이트
|
v
5. 결과 공개 게시 + 플레이어에게 DM 완료 알림
|
v
6. Adventures 탭에 내역 자동 기록

모든 명령은 무사수행 봇 계정을 멘션한 뒤 대괄호 `[ ]` 안에 입력합니다.
